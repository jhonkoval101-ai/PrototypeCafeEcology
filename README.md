## 🎮 Описание
Прототип экологического симулятора: игрок управляет кафе, балансируя между прибылью и загрязнением окружающей среды.

## ⌨️ Управление
| Действие | Клавиша |
|----------|---------|
| Перемещение | WASD / Стрелки |
| Взаимодействие | E / ЛКМ |
| Бросить предмет | ПКМ |
| Пауза | Esc |

## 🛠️ Технологии

- C#

## 🚀 Сборка
1. Открыть сцену `Assets/Scenes/MainScene.unity`
2. Запустить через Play Mode в редакторе
3. Или собрать: File → Build Settings → Windows → Build
